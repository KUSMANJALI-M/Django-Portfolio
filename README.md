# Django Portfolio Website

A personal portfolio website built using Django to showcase my projects, skills, and experience.

## Features

* Responsive portfolio design
* About Me section
* Projects showcase
* Contact section
* Resume download option
* GitHub profile integration

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* Bootstrap

## Installation

1. Clone the repository:

   git clone https://github.com/KUSMANJALI-M/Django-Portfolio.git

2. Navigate to the project directory:

   cd Django-Portfolio

3. Install dependencies:

   pip install -r requirements.txt

4. Run migrations:

   python manage.py migrate

5. Start the development server:

   python manage.py runserver

6. Open your browser and visit:

   http://127.0.0.1:8000/
